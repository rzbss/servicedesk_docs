##Keyboard shortcuts

###Global Shortcuts

Go to Home tab:
```
g h
```
Go to Requests tab:
```
g r
```
Go to Request:
```
alt+r
```
Create New Request:
```
n r
```
Search: 
```
/
```
Cancel Cursor focus from fields:
```
esc
```
 

**List View Actions (All Modules)**

```
    [
    Hide/Show Left Panel
    s
    Open List Search
    v
    Open Views Menu
```
